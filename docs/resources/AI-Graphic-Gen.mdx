# AI Graphic Generation

#### How to create post graphics with AI help?

It's very easy, just copy paste to ChatGPT the entire documentation at https://docs.orivonstack.com/docs/orivon, then copy the entire post you made, using this format:


```
Orivon Documentation:
[Copy paste of the docs]


The post I want to publish:

[Copy paste of your post]


Create me a design picture to use for that post, horizontal and for Twitter
```

It may add a fake Orivon logo, in that case you can replace it with the use of an editor like pixlr https://pixlr.com/editor/

Otherwise, you can prevent it at all, just add in the prompt:  
`Do not use any logo regarding Orivon`
